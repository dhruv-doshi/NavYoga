Yoga Pose Estimation

This project aims to build a web-based yoga pose estimation and correction application that uses the device’s camera to analyze body posture in real time and provide immediate corrective feedback. The application runs directly in a web browser, allowing users to access guided posture feedback without installing additional software.
The system uses MediaPipe Pose along with the BlazePose to detect key body landmarks from live webcam input. These landmarks represent important joints such as the shoulders, elbows, hips, knees, and ankles. The application processes these landmarks to calculate joint angles and evaluate body alignment during yoga practice.
To support yoga pose correction, the system compares the user’s joint angles with predefined reference ranges for specific yoga poses with a certain error range. When deviations from the correct posture are detected, the application generates real-time corrective suggestions, such as instructing the user to straighten an arm, bend a knee further, or adjust body alignment. These corrections are displayed as on-screen messages, helping users refine their posture dynamically.
The interface overlays a visual skeleton on top of the live video feed, allowing users to clearly see how their body is being interpreted by the system. Incorrect joints or misaligned body parts can be visually highlighted, making posture errors easier to identify and fix.
